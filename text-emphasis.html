The difference between the <i> element and the <em> element.
The difference between the <b> element and the <strong> element.

The <i> denotes a change in voice or mood.

The <em> denotes text that needs to be emphasized.

The <b> drawing attention to the text, but not indicating any extra importance.

The <strong> emphasizes the emphasis of the text, meant to draw attention.

For situations where the text is modified only for presentational purposes, CSS should be used.
